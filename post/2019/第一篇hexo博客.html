终于…

经历了不靠谱的BSP，自建的Typecho也是命运多舛； 最后还是决定用github的托管加上hexo静态生成博客； 保留所有博客的md源文件同步到网盘； 简单的永恒的就是最好的，写写字而已，花俏的功能也没什么用； 刚装好，第一篇权当测试，水了

感谢：

  • Hexo，安装真的炒鸡简单；
  • 最良心的同性博客托管商：Github；
  • 最牛逼的markdown编辑器：Typora；

记录一下，Hexo在Windows10下安装：

  1. 装Git和nodejs：
  • https://git-scm.com/download/win
  • https://nodejs.org/en/
  1. 建个目录，然后：
1
2
3
4
5

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
  1. 常见操作：
  • Create a new post
1

$ hexo new "My New Post"

More info: Writing

  • Run server
1

$ hexo server

More info: Server

  • Generate static files
1

$ hexo generate

More info: Generating

  • Deploy to remote sites
1

$ hexo deploy
